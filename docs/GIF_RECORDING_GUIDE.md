# GIF 录制指南

本文档指导您如何录制项目演示 GIF。

## 推荐工具

### Windows
- **ScreenToGif** (免费开源)
  - 下载: https://www.screentogif.com/
  - 特点: 轻量、可编辑、导出质量高

### macOS
- **Kap** (免费开源)
  - 下载: https://getkap.co/
  - 特点: 简洁、高质量

### 跨平台
- **LICEcap** (免费)
  - 下载: https://www.cockos.com/licecap/
  - 特点: 简单易用

## 需要录制的场景

### 1. 交互式提示词生成流程
**文件名**: `demo-interactive-flow.gif`

**录制步骤**:
1. 打开应用首页
2. 在输入框输入: "帮我写一篇关于人工智能的文章"
3. 点击发送
4. 等待 AI 显示"优化方向建议"表单
5. 选择几个选项（展示单选/多选切换）
6. 点击"生成最终 Prompt 文档"
7. 显示最终生成的结构化提示词

**时长**: 15-20秒
**分辨率**: 1280x720 或更高

### 2. 文件上传与解析
**文件名**: `demo-file-upload.gif`

**录制步骤**:
1. 打开应用首页
2. 点击输入框左侧的"📎"图标（或直接拖拽文件）
3. 选择一个 PDF 或图片文件上传
4. 等待文件解析完成（显示文件预览）
5. 输入相关问题，如："总结这个文档的要点"
6. 点击发送
7. 显示 AI 基于文件内容的回复

**时长**: 20-25秒
**分辨率**: 1280x720 或更高

### 3. 自定义模板管理
**文件名**: `demo-template-management.gif`

**录制步骤**:
1. 打开应用首页
2. 点击右上角的"⚙️"设置图标
3. 滚动到"系统提示词"部分
4. 在模板选择器中选择"默认模板"
5. 修改系统提示词内容
6. 输入新模板名称（如："技术文档专用"）
7. 点击"保存为新模板"按钮
8. 在下拉列表中看到新模板出现
9. 切换到新模板，显示内容已更新
10. 点击模板旁边的删除按钮（演示删除功能）

**时长**: 25-30秒
**分辨率**: 1280x720 或更高

## 录制技巧

### 准备工作
1. **清理浏览器**: 关闭不必要的标签页和扩展
2. **调整窗口大小**: 建议使用 1280x720 或 1920x1080 分辨率
3. **准备测试数据**: 提前准备好要输入的文本和上传的文件
4. **检查 API 配置**: 确保 API Key 已配置且可用

### 录制设置
- **帧率**: 15-20 FPS（足够流畅且文件不会太大）
- **质量**: 中等到高（ScreenToGif 建议 80-90%）
- **循环**: 设置为循环播放
- **延迟**: 每个操作之间留 1-2 秒让用户看清

### 优化建议
1. **减小文件大小**:
   - 删除不必要的帧
   - 降低颜色数（256 色通常足够）
   - 使用 GIF 优化工具（如 gifsicle）

2. **提升观看体验**:
   - 鼠标移动要平滑
   - 避免快速滚动
   - 重要操作可以稍微放慢
   - 最终文件大小控制在 5MB 以内

## 注意事项

⚠️ **隐私保护**:
- 录制前清空聊天历史
- 不要展示真实的 API Key
- 使用示例数据而非真实敏感信息

✅ **质量检查**:
- 录制完成后完整播放一遍
- 确保所有关键步骤都清晰可见
- 检查文字是否清晰可读
- 确认 GIF 能正常循环播放
